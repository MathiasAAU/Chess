Chess game: Github
How to play:
  1. Click Game in the top right
  2. White player moves first then black
  3. Play until one side wins
  4. A new game can be started at any point
Missing features:
  • Detecting when the game state is a draw or a stalemate
  • The ability to forfeit a game
  • Displaying A to H for the files and 1 to 8 for the ranks
  • The ability to see more the 17 moves back and for a move displaying if a piece was killed
  • Keep track of a player's wins and the option to switch color
  • Fixing bugs that happends when a king is in check (E.g being able to move a piece you should not be able to)
